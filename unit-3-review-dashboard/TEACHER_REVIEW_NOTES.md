# Unit 3 Review Dashboard - Teacher Review Notes

The dashboard was built from the Unit 3 study guide packet and teacher key.

## Source Files

- Student packet: `Unit 3 Test Study Guide - New Format ITERATION 2.pdf`
- Teacher key: `Unit 3 Test Study Guide - New Format ITERATION 2 KEY.docx`

The source PDF, DOCX, and answer key files were not changed.

## Included Problems

The app includes all 24 worked study guide entries extracted from the teacher key.

The packet page preview uses rendered images from the 12-page student study guide.

## Similar Practice

The similar-practice section now generates outside-the-packet practice instead of routing students to the matching Version A / Version B companion problem.

Each major Unit 3 skill has at least four similar prompts available through the `New Similar Problem` button:

- domain and range from graph descriptions
- function values and inputs
- slope from two points
- linear rules from graphs, tables, and descriptions
- function/not-a-function decisions across representations
- linear context graph situations

## Answer-Key Notes

No unresolved answer-key issues were found during the automated build. Answers and worked steps came from the teacher key.

The typed checker for similar-practice items is intentionally lightweight. It accepts common compact formats, but equivalent expression, graph, coordinate, interval, or inequality notation may still need students to reveal the answer and compare manually.
