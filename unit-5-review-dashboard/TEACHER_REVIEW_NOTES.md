# Unit 5 Review Dashboard - Teacher Review Notes

The dashboard was built from the Unit 5 study guide packet and teacher key.

## Source Files

- Student packet: `Unit 5 Test Study Guide - New Format DRAFT.pdf`
- Teacher key: `Unit 5 Test Study Guide - New Format DRAFT KEY.docx`

The source PDF, DOCX, and answer key files were not changed.

## Included Problems

The app includes all 32 worked study guide entries extracted from the teacher key.

The packet page preview uses rendered images from the 9-page student study guide.

## Similar Practice

The similar-practice section now generates outside-the-packet practice instead of routing students to the matching Version A / Version B companion problem.

Each major Unit 5 skill has at least four similar prompts available through the `New Similar Problem` button:

- checking whether an ordered pair solves a system
- solving two linear equations by setting expressions equal
- classifying systems by number of solutions
- setting up substitution and choosing elimination moves
- writing systems from contexts
- testing and identifying solution regions for systems of inequalities
- interpreting solutions in context
- solving systems by substitution and elimination
- solving context systems and coin problems

## Answer-Key Notes

No unresolved answer-key issues were found during the automated build. Answers and worked steps came from the teacher key.

The typed checker for similar-practice items is intentionally lightweight. It accepts common compact formats, but equivalent expression, graph, coordinate, interval, or inequality notation may still need students to reveal the answer and compare manually.
