# Unit 7 Quest Review Dashboard - Teacher Review Notes

The dashboard was built from the Unit 7 study guide packet and teacher key.

## Source Files

- Student packet: `Unit 7 Quest Study Guide - New Format DRAFT.pdf`
- Teacher key: `Unit 7 Quest Study Guide - New Format DRAFT KEY.docx`

The source PDF, DOCX, and answer key files were not changed.

## Included Problems

The app includes all 24 worked study guide entries extracted from the teacher key.

The packet page preview uses rendered images from the 7-page student study guide.

## Similar Practice

The similar-practice section now generates outside-the-packet practice instead of routing students to the matching Version A / Version B companion problem.

Each major Unit 7 quest skill has at least four similar prompts available through the `New Similar Problem` button:

- y-intercepts, x-intercepts, vertices, axes of symmetry, and opening direction
- solving quadratics by factoring
- solving with the square root property
- identifying quadratic formula coefficients
- solving with the quadratic formula
- combining graphing features for quadratics

## Answer-Key Notes

No unresolved answer-key issues were found during the automated build. Answers and worked steps came from the teacher key.

The typed checker for similar-practice items is intentionally lightweight. It accepts common compact formats, but equivalent expression, graph, coordinate, interval, or inequality notation may still need students to reveal the answer and compare manually.
