# Final Exam Dashboard - Teacher Review Notes

The app preserves `Final-Exam-Study-Guide.pdf` unchanged and builds the student dashboard beside it.

## Supplemental Key Incorporated

The supplemental answer key in `supplemental_answer_key.docx` has been used to update the graph-heavy, scanned, and corrected answer entries in `script.js`.

Updated items include:

- Domain/range graph items: 7-14.
- Corrected scanned or hard-to-read items: 16, 23, 27, 32, 36, 39, 47-49, 51-53, 56, 62-65, 67-70, 72-75, 81-82, 86-87, 103, 112, 130-131.
- Visual multiple-choice items: 134-137, 141, 143-145, 149-150, 153-156.

The app no longer marks those updated items with the "Teacher check" badge.

## Canvas Use

This is a static app made of:

- `index.html`
- `styles.css`
- `script.js`
- `packet-pages/`

For Canvas, host or upload the whole folder so the page images remain available. The app uses MathJax from a CDN for math rendering, so students need internet access when using the page.

## Student Workflow

1. Students complete problems on their paper packet first.
2. Students click the problem number.
3. They reveal the final answer first.
4. They reveal steps only after checking their own work.
5. They can mark the problem checked and open similar practice with answer checking.
