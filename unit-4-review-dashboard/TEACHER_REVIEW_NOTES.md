# Unit 4 Review Dashboard - Teacher Review Notes

The dashboard was built from the Unit 4 study guide packet and teacher key.

## Source Files

- Student packet: `Unit 4 Study Guide - New Format ITERATION 2.pdf`
- Teacher key: `Unit 4 Study Guide - New Format ITERATION 2 KEY.docx`

The source PDF, DOCX, and answer key files were not changed.

## Included Problems

The app includes all 46 worked study guide entries extracted from the teacher key.

The packet page preview uses rendered images from the 24-page student study guide.

## Similar Practice

The similar-practice section uses the matching Version A / Version B companion problem whenever one exists. This keeps the extra practice aligned with the packet structure students already see.

## Answer-Key Notes

No unresolved answer-key issues were found during the automated build. Answers and worked steps came from the teacher key.

The typed checker for similar-practice items is intentionally lightweight. It accepts common compact formats, but equivalent expression, graph, coordinate, interval, or inequality notation may still need students to reveal the answer and compare manually.
